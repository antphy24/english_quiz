# 📘 EnglishScore C1 Training Quiz

A mobile-optimized, interactive web application designed for English language learners transitioning from **B2 (Upper Intermediate)** to **C1 (Advanced)** levels[cite: 1]. This quiz focuses on advanced collocations, inversions, and complex grammatical structures[cite: 1].

## 🚀 Live Demo
You can view the project live here: [https://antphy24.github.io/english_quiz/](https://antphy24.github.io/english_quiz/)[cite: 1]

## ✨ Key Features

- **Mobile-First Design:** Large touch targets and responsive layout optimized for smartphones[cite: 1].
- **Real-Time Feedback:** Instant color-coded validation (Green for correct, Red for incorrect) with detailed explanations for every answer[cite: 1].
- **Progress Tracking:** Dynamic progress bar and question counter to keep users engaged[cite: 1].
- **Live Scoring & Timer:** Track your accuracy and speed simultaneously[cite: 1].
- **Advanced Curriculum:** 20 hand-picked questions covering C1 essentials like:[cite: 1]
    - Inversions (e.g., "Rarely have we seen...")[cite: 1]
    - Advanced Collocations (e.g., "Make a decision" vs "Take an impact")[cite: 1]
    - Verb Patterns (e.g., "Suggest doing" vs "Avoid answering")[cite: 1]
    - Mixed Conditionals[cite: 1].

## 🛠️ Technology Stack

- **HTML5:** Semantic structure for accessibility and SEO[cite: 1].
- **CSS3:** Custom properties (CSS variables), Flexbox, and modern UI/UX patterns for a "native app" feel[cite: 1].
- **JavaScript (ES6+):** Vanilla JS for state management, timer logic, and dynamic DOM manipulation[cite: 1].
